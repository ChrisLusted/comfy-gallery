# CMS Gallery
Author: [The Working Group](http://www.theworkinggroup.ca)

---

## What is it?

CMS Gallery is an easy to use, super simple gallery plugin for Comfortable Mexican Sofa.
  
  
## Install

### 1. Add the gem definition to your Gemfile:
    
    config.gem 'cms_gallery'
    
### 2. From within your Rails project run:
    
    bundle install
